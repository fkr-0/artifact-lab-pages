ASTER RELAY

Nightline / midnight indigo / pixel-light / restrained neon

Aster Relay is a quiet, expressive identity for a tech-education collective: civic-minded, practical, protective, and just a little more alive at the edges.

This revision starts from the uploaded Midnight Galaxy reference and pulls it into a distinct sibling system with deeper indigo fields, cleaner violet structure, tiny graffiti traces, pixel affection, and a super subtle neon signal. The base stays readable.

position

Stance and design philosophy

What it is

A mixed-gender tech collective identity built for workshops, documentation, release notes, letters, social cards, and tools related to practical self-protection and mutual literacy.

What it is not

Not tactical cosplay. Not glossy startup futurism. Not nostalgia merch. The system remains calm and exact even when it carries a little hand-made residue.

The Nightline edge

Graffiti whisper: slanted underlines, spray-soft traces, tiny tags.

Pixel affection: stepped stars, dither fields, visible grid logic.

Neon punch: cyan or magenta only as pulse colors, never as a flood.

Proof words

steady · literate · protective · lived-in · non-performative

Writing posture

Write like a capable peer. Use concrete nouns and verbs. Let posters carry attitude; let docs carry trust. One sharp phrase is enough.

configurability

Name can be swapped without breaking the system

Files added

config/brand.json
scripts/apply_brand.py
templates/configurable/*

Change the collective name, slug, line, domain, email, and document prefix in one place.

How to use it

1. Edit the JSON config.
2. Run the script.
3. Use the rendered outputs in generated/.

The symbol system stays stable while the naming layer adapts.

Why this matters

Collectives evolve. Names change. Programs split or merge. Making the naming layer configurable protects the investment in the visual system and keeps the pack reusable for future iterations.

Use the edge sparingly

The strongest version of this identity is still quiet. The graffiti trace should feel remembered, not performed. The pixel language should feel loved, not gimmicky. The neon should feel like a pulse, not a flood.

That balance is what keeps the system modern, readable, and durable across code, docs, letters, sites, and collective releases.
