# Aster Relay — Quickstart

## Core rule

Keep the base quiet. Add edge only where it helps recognition or warmth.

## Palette split

- **Base:** obsidian, night, indigo
- **Primary energy:** relay violet
- **Readable structure:** mist and ice
- **Rare emphasis:** neon cyan and magenta

## When to use the new edge

Use the graffiti / pixel / neon layer for:
- event posters
- social cards
- workshop visuals
- release covers
- highlight sections in docs or sites

Keep it lighter in:
- letters
- proposals
- long-form docs
- official statements

## Configurable name

Edit `config/brand.json` and run:

```bash
python scripts/apply_brand.py
```

Generated outputs go into `generated/`.
