# Code Header Templates

```txt
// {BRAND_NAME} / module
// Summary: {what this module does}
// Interfaces: {entry points}
// Constraints: {performance, privacy, or compatibility constraints}
```

## Script file

```txt
// {BRAND_NAME} / utility script
// Inputs: {files, args, env vars}
// Outputs: {artifacts, logs, side effects}
// Failure mode: {what happens on partial failure}
```
