# Configurable Brand System

This pack ships with a sample collective name: **Aster Relay**.

To change it:

1. Edit `config/brand.json`
2. Run `python scripts/apply_brand.py`
3. Use the rendered files in `generated/`

## What is configurable now

- brand name
- slug
- line
- descriptor
- domain
- contact email
- document prefix

## Design note

The visual mark is name-agnostic, so the strongest benefit comes from updating the
lockups, letterhead, and writing templates while keeping the same symbol system.
