# Aster Relay Identity Pack — Nightline Edge

Aster Relay is a proposed collective identity for a tech-education group focused on
practical self-protection, mutual literacy, and calm technical competence.

This revision keeps the earlier midnight foundation from the uploaded **Midnight Galaxy**
reference, but adds a restrained new edge:
- a **simple 90s graffiti whisper** rather than loud streetwear branding
- a visible **love of pixel art** through stepped stars, dither, and grid rhythm
- a **super subtle neon punch** through controlled cyan/magenta signal highlights

## What is inside

- `brand/` — brand book, quickstart, language guide
- `assets/logo/` — updated mark and lockups
- `assets/icons/` — six pixel-light icon concepts
- `assets/art/` — revised posters, social card, wallpaper, interactive HTML art
- `tokens/` — CSS, SCSS, JSON, and VS Code theme
- `templates/` — Markdown, docs CSS, letterhead, LaTeX, comms, code comment presets
- `config/brand.json` — single source of truth for the brand name and metadata
- `scripts/apply_brand.py` — renders configurable templates with a new name

## Fastest path

1. Edit `config/brand.json`
2. Run `python scripts/apply_brand.py`
3. Use the files written to `generated/`

The sample outputs in this pack use **Aster Relay** as the default configured name.

## Identity summary

- Name: **Aster Relay**

- Descriptor: **practical privacy and mutual tech literacy**
- Prose width: `72ch`

## The Nightline edge

The edge is deliberate but restrained. It should feel like civic-tech material that has spent a
little time near posters, keyboards, notebooks, and walls — not a fashion drop, not a rave flyer,
and not a retro parody.
